# Creative Developer Portfolio 🚀

A modern, highly interactive, and visually stunning developer portfolio built to showcase projects, skills, and professional experience. 

![Portfolio Preview](./profile.png) <!-- Update with an actual screenshot of the portfolio -->

## ✨ Key Features

- **Cinematic 3D Background:** Powered by Three.js, featuring a starfield, diverse 3D geometries (Torus Knots, Dodecahedrons), and dynamic lighting that reacts to mouse movement.
- **Mandatory "Warp" Entry:** A cinematic landing experience that requires the user to press "Space" to start, triggering a hyper-speed warp animation before revealing the content.
- **Premium Custom Cursor:** A sleek, glowing dot with a smooth, trailing follower that scales up and changes color when hovering over interactive elements.
- **Glassmorphism Design:** Beautiful, frosted-glass cards throughout the UI (`.glass-card`), providing a modern and clean aesthetic over the dynamic background.
- **Responsive Layout:** fully optimized for both desktop and mobile viewing.
- **Project Showcase:** Detailed project cards with hover effects and direct links to GitHub repositories and live demos.
- **Timeline Experience:** A structured, vertical timeline layout to display educational and professional history.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, complex animations (`@keyframes`), and backdrop filters for glassmorphism.
- **JavaScript (Vanilla):** DOM manipulation, intersection observers for scroll animations, custom cursor logic, and state management.
- **Three.js:** WebGL library used to render the complex, interactive 3D background and lighting effects.

## 🚀 Getting Started

Since this is a static frontend project, no build tools or package managers are required to run it locally.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mauryababita/portfolio.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd portfolio
   ```
3. **Run the project:**
   Simply open the `index.html` file in your preferred modern web browser. 
   *(Note: For the best experience, you can use a local server like VS Code's "Live Server" extension to avoid any local CORS issues with external assets, though it's not strictly necessary).*
